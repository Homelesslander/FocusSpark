# ✨ FOCUS SPARK
### *Empowering Neurodivergent Youth Through Gamified Task Management*

---

## 🌟 What is Focus Spark?

**Focus Spark** is a friendly, neurodivergent-designed web app that transforms daily routines into an interactive and rewarding experience. We combine gentle gamification with clear visual progress tracking to help neurodivergent adolescents manage tasks with confidence and ease.

> *Because every task completed deserves celebration, and every neurodivergent mind deserves support.*

## 🎯 Core Features

| 🧠 **Feature** | 🎭 **Challenge Addressed** | 🛠️ **How It Works** | ✨ **Impact** |
|---|---|---|---|
| **💭 Feelings Thermometer** | Difficulty identifying and expressing emotions | Visual emotion scale with calming space options | Supports emotional regulation and recognition; provides sensory-safe coping strategies |
| **📋 Visual Task Cards** | Challenges with task planning and independent completion | Structured, visual task layouts that break work into manageable steps | Transforms overwhelming tasks into achievable steps; supports organization and independence |
| **⏱️ Visual Timer** | Difficulty understanding time limits and pacing tasks | Clear, visual countdown display for task duration | Improves time awareness; helps complete tasks within set periods |
| **📊 Progress Tracker** | Reduced motivation and difficulty with consistent participation | Gamified reward system with visible progress feedback | Encourages engagement; provides motivational feedback and celebrates achievements |
| **♿ Accessible Interface** | Sensory sensitivities and cognitive overload | Simple design with high contrast, clear navigation, and minimal clutter | Reduces cognitive load; improves usability across devices; supports neurodivergent access needs |

## 🎮 How Focus Spark Supports Neurodivergent Needs

### Emotional Regulation 💙
- **Feelings Thermometer** helps identify and express emotions safely
- Customizable calming spaces with sensory-friendly resources
- Visual emotion cues reduce frustration and anxiety

### Task Organization 📝
- **Visual Task Cards** break complex tasks into clear, manageable steps
- Structured layouts reduce decision fatigue
- Step-by-step guidance supports independence

### Time Awareness ⏰
- **Visual Timer** makes time concrete and understandable
- Helps learners pace themselves effectively
- Reduces anxiety around task completion deadlines

### Motivation & Engagement 🎯
- **Progress Trackers** and reward systems celebrate small wins
- Gamification creates positive reinforcement loops
- Visual feedback builds confidence and sustained effort

### Accessibility First ♿
- Designed with sensory sensitivities in mind
- High contrast, clear fonts, minimal animations
- Simple navigation reduces cognitive load
- Works seamlessly across all devices

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- No installation required!

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Homelesslander/Nuerodivergent-Support-Portal.git
   cd Nuerodivergent-Support-Portal
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000`

## 📱 Features in Detail

### 1. Feelings Thermometer 💭
Emotional self-regulation made visual. Users identify their emotional state on a spectrum and receive personalized calming strategies tailored to neurodivergent needs.

### 2. Visual Task Cards 📋
Complex tasks simplified into visual, sequential steps. Each card clearly shows what needs to be done, reducing overwhelm and supporting independent task completion.

### 3. Visual Timer ⏱️
Time becomes tangible. Watch progress unfold with a clear, visual countdown that helps users understand pacing and stay on task.

### 4. Progress & Reward System 🏆
Celebrate every milestone! Visual progress trackers and achievable rewards keep motivation high and make success visible.

### 5. Autism-Friendly Design ♿
Everything is designed with neurodivergent brains in mind:
- Clear visual hierarchy
- High contrast options
- Minimal sensory overload
- Intuitive, predictable navigation

## 🎨 Design Philosophy

Focus Spark was built on a foundation of **neurodivergent-affirming design principles**:

✨ **Clear & Predictable** - Reduce decision fatigue with straightforward layouts  
💙 **Sensory-Friendly** - Thoughtful use of color, sound, and animation  
🧠 **Cognitively Accessible** - Simple language, visual supports, minimal clutter  
🎯 **Strength-Based** - Celebrate abilities and build on existing skills  
🤝 **Community-Centered** - Designed *with* neurodivergent voices, not *for* them

## 📖 How It Works

1. **Sign Up** - Create your account in seconds
2. **Add Tasks** - Break down daily activities into manageable steps
3. **Visual Supports** - Use the Thermometer, Timer, and Task Cards
4. **Track Progress** - Watch your accomplishments grow
5. **Earn Rewards** - Celebrate milestones and stay motivated
6. **Reflect** - Learn what strategies work best for you

We believe in co-design with neurodivergent communities. Your input matters!

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

### Accessibility Checklist for Contributors
- [ ] Code follows WCAG 2.1 AA standards
- [ ] Changes tested with screen readers
- [ ] Visual changes tested with high contrast mode
- [ ] No motion/animation that could trigger sensory overload
- [ ] Documentation is clear and uses plain language
