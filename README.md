<ins>**FOCUSSPARK**</ins>

Focus Spark is a friendly web app designed to help Nuerodivergent persons manage tasks through gentle gamification and clear visual progress. 
This platform transforms daily routines into an interactive and rewarding experience for adolescents with neurodivergence.
